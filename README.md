
Point to a URL, template the payload, set a wordlist, and launch.

```
Usage of ./gbrd:
  -continue
    	continue from last saved progress
  -form string
    	form template with {{USERNAME}} and {{PASSWORD}} placeholders (required)
  -passpath string
    	path to password list file (required)
  -target string
    	target endpoint url (required)
  -username string
    	single username to test
  -userpath string
    	path to username list file
```
